Kantele application generator.

## Installation

```sh
$ npm install -g kantele
```

## Quick Start

Create an app:

```bash
$ kantele app /tmp/foo

```

Create an app in coffeescript:

```bash
$ kantele app -c /tmp/foo
```

Install dependencies:

```bash
$ cd /tmp/foo && npm install
```
